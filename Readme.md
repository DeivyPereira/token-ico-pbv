## Deployment

Para implementar este proyecto, ejecute

```bash
  npm install
```
Al instalar todos los paquetes tendra Truffle que es el encargado de compilar y migrar el contrato desarrollado y generara un JSON con toda la informaciÃ³n a utilizar
puede iniciar el proyecto automaticamente con el comando
```bash
  npm start
```
y podras interacturar con el Deployment de Rinkeby en el cual solo debes tener credito para usarlo .



## Deployment Local

Para implementar este proyecto en local es necesario tener - [Ganache](https://www.trufflesuite.com/ganache) instalado para que funcione como Blockchain local
luego compilar y deployment el proyecto

```bash
  npm install
```
Compilar en Truffle
```bash
  truffle compile
```
Deployment en local
```bash
  truffle migrate
```
Funcionamiento
Deployment en local
```bash
  npm start
```
